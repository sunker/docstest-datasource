# Configuration

This section covers everything you need to set up the DocsTest data source in Grafana, from adding a new data source instance to provisioning it via configuration files.

## Before you begin

Make sure you have:

- Administrator access to your Grafana instance
- A [Service Name] account with the necessary [permissions](./permissions/)
- The API endpoint URL for your [Service Name] instance
