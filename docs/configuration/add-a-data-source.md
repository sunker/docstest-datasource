# Add a data source

To add a new DocsTest data source instance:

1. In Grafana, navigate to **Connections** > **Data sources**.
1. Click **Add new data source**.
1. Search for **DocsTest** and select it.
1. Configure the connection settings as described in [Configuration](./configuration/).
1. Click **Save & test** to verify the connection.

If the test fails, check the [Troubleshooting](../troubleshooting/) section for common issues.
