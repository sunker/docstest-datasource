# Query editor

The DocsTest data source provides two ways to build queries:

- **[Query builder](./query-builder/)** - A visual interface for constructing queries without writing code. Best for common query patterns and users unfamiliar with the query syntax.
- **[Code editor](./code-editor/)** - A text-based editor for writing queries directly. Use this for advanced queries, complex filters, or when you need full control over the query syntax.

You can switch between modes at any time using the toggle in the query editor header. Switching from query builder to code editor preserves your query. Switching from code editor to query builder is only possible when the query can be parsed into the visual format.
