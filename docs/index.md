# Overview

<!-- Describe the problem this plugin solves and the value it provides. -->

The DocsTest data source plugin connects Grafana to **[Service Name]**, enabling you to query, visualize, and alert on your data directly within Grafana dashboards.

Use this plugin when you need to:

- Monitor [Service Name] metrics alongside other data sources in Grafana
- Build dashboards that combine [Service Name] data with other operational data
- Set up alerts based on [Service Name] query results

## Key features

<!-- List the main capabilities of the plugin. -->

- **Query builder** - Build queries visually without writing raw query syntax
- **Code editor** - Write queries directly for full control and advanced use cases
- **Annotation support** - Overlay events from [Service Name] on any Grafana panel
- **Template variables** - Use Grafana template variables for dynamic, reusable dashboards
- **Alerting** - Create alert rules based on query results from [Service Name]

## Requirements

<!-- Specify any version requirements or dependencies. -->

- Grafana **10.0** or later
- A [Service Name] account with API access enabled
- Network access from the Grafana instance to the [Service Name] API endpoint

## Getting started

1. [Install and configure](./configuration/) the data source
2. [Build queries](./query-editor/) using the query builder or code editor
3. Create dashboards and alerts using your data
